# Reproduction de CamemBERT-bio sur le NER médical français
Reproduction des résultats du papier CamemBERT-bio (Touchent & de la Clergerie, Inria) sur la reconnaissance d'entités nommées médicales avec le corpus QUAERO. Travail en cours — fine-tuning de camembert-base vs camembert-bio, évaluation F1 avec seqeval.
